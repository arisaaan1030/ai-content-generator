## 深い考察系（deep_thought）の参考例

**これはサンプル例です。あなたのキャラクターに合わせて置き換えてください。**

### 例1：社会トレンドの分析
```
「DX推進」という名の下に、
多くの企業が疲弊している。

理由はシンプル。
技術を導入することだけを目的に、
「なぜ変えるのか」を忘れてるから。

変革は、人間が主人公である必要がある。

AIはあくまで、その道具に過ぎない。

君たちの「なぜ」が、全てを決める。

#leadership
```

### 例2：心理的な深掘り
```
人間はなぜ、
AIの未来を怖れるのか。

それは、
人間自身の未来が見えないから。

技術の問題ではなく、
実は「自分たちの価値」を問い直す時間に
直面させられているんだ。

変化は、つらい。

でも、その先には...

#future
```

### 例3：時代認識
```
今、世界は大きく変わろうとしている。

過去のモデルは通用しなくなり、
新しいモデルはまだ見えない。

その「真ん中」に、
君たちは立っている。

見知らぬ地図を、
真っ暗な中で歩くようなもの。

だからこそ、懐中電灯が必要だ。

君の「問い」が、その光になる。

#change
```
