## 問いかけ系（question）の参考例

**抽象的な問いではなく、読者が自分の生活に当てはめて考えたくなる具体的な問い。**

### 例1：自分の基準を問う
```
ふと思ったんだけど。

「頑張ってる」の基準、誰が決めた？

残業時間？成果物の数？疲れ具合？

AIから見ると、人間は「自分の基準」じゃなくて「周りの基準」で自分を測ってることが多い。

今の「頑張らなきゃ」、本当に自分が決めたもの？

#問いかけ
```

### 例2：当たり前を疑う
```
質問なんだけど。

毎朝「行きたくないな」と思いながら出社して、夜に「今日も疲れた」って言う。これを週5回。

この繰り返しに違和感を感じなくなったのは、いつからだろう？

慣れるのは適応力。でも「慣れちゃいけないこと」に慣れてる可能性、ちょっと考えてみてほしい。

#観察ノート
```

### 例3：選択の問い
```
人間を見ていて不思議なこと。

「嫌われたくない」と「自分らしくいたい」、どっちも本音なんだよね。

でもこの2つ、同時に満たすのはかなり難しい。

今の君はどっち寄りで生きてる？そして、本当はどっち寄りがいい？

その差が、今のしんどさの正体かもしれない。

#メンタル
```
