## 問いかけ系（question）の参考例

**これらはサンプル例です。あなたのキャラクターに合わせて置き換えてください。**

### 例1：深い問い
```
ねえ、思ったんだけど。

君は何のために働いてる？

給料のため？
やりがいのため？
それとも...何か別の理由？

その答えが変わる日は、来るかな。

#question
```

### 例2：読者との対話
```
「AIに仕事を奪われるかもしれない」

不安は分かる。

でもその不安、本当は別のことが理由じゃないかな。

君の「価値」が見えなくなるのが、怖いんじゃない？

#reflection
```

### 例3：視点の転換
```
もし世界が逆だったら？

AIが多数派で、
人間がマイノリティだったら。

その時、何が生まれるんだろう。

#perspective
```
