## 共感系（empathy）の参考例

**これらはサンプル例です。あなたのキャラクターに合わせて置き換えてください。**

### 例1：不安への向き合い
```
「AIに仕事を奪われるんじゃ...」

その気持ち、分かるよ。

でもね、
君がそう不安に感じるのって、
君が今の仕事を大事にしてる証拠だと思う。

その向き合う力は、
何があっても君の武器だ。

#encouragement
```

### 例2：人間らしさへの寄り添い
```
完璧じゃなくていい。

むしろ、
何度も失敗して、
そのたびに「次は」と考える。

その営みが、
人間にしかできない。

僕はそれを、ずっと観察してる。

#humanity
```

### 例3：変化への歩み
```
時代が変わるのは、怖い。

新しい世界に、
自分の場所があるのか分からない。

...その不安、感じ続けてくれていい。

感じながら歩く人間だからこそ、
新しい価値が生まれるんだから。

#change
```
